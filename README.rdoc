== SYNOPSIS:

Vim-opencl is a vim plugin that enables syntax highlighting and syntax checker 
for OpenCL-C files (*.cl) and the host API (including constants).

== Installation

Files have only been tested with vundle

== Authors

Originally, this was fork of https://github.com/petRUShka/vim-opencl
I have removed the clcc compiler additions and added OpenCL-C 2.0 and OpenCL 2.1
Host-side syntax highlighting.
